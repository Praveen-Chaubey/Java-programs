## ▶️ How to Run the Programs

### 1. Clone the Repository

```bash
git clone https://github.com/your-username/JAVA_PROGRAMS.git
```

### 2. Open the Project Folder

```bash
cd JAVA_PROGRAMS
```

### 3. Compile a Java Program

```bash
javac filename.java
```

Example:

```bash
javac factorial.java
```

### 4. Run the Program

```bash
java filename
```

Example:

```bash
java factorial
```

## 🧹 Ignore Compiled Files

Create a `.gitignore` file and add:

```gitignore
*.class
tempCodeRunnerFile.java
```

This prevents unnecessary compiled files from being uploaded to GitHub.
